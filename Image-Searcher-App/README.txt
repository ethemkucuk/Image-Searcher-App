# Image Search App

Bu proje, kullanıcıların belirli anahtar kelimelerle unsplash API üzerinden görsel araması yapmasına olanak tanır ve bulunan görselleri görüntüler.

## Özellikler

- Kullanıcı, arama kutusuna anahtar kelime yazarak belirli bir konu hakkında görsel arayabilir.
- Görsel sonuçları sayfa sayfa görüntülenebilir.
- Her görsel sonucu, resmin küçük bir önizlemesi ve bağlantısı ile birlikte gösterilir.
- "Daha Fazla Göster" düğmesine tıklanarak daha fazla sonuç yüklenebilir.

## Kullanım

1. Anahtar kelimeyi arama kutusuna yazın.
2. "Search" düğmesine tıklayarak aramayı başlatın.
3. Görsel sonuçlarını inceleyin.
4. Daha fazla sonuç yüklemek için "Daha Fazla Göster" düğmesine tıklayın.

## Kurulum

1. Bu projeyi bilgisayarınıza klonlayın veya indirin.
2. `index.html` dosyasını tarayıcınızda açarak uygulamayı kullanın.
